OBMS

6-week training project


OBMS stands for Online Building Material Store. It is the online portal of a building material store situated in jamalpur. The general purpose of the project is provide a estimated cost that they should spent for for the goods that they want to buy. Also if they want to buy then they can from the given site.

Technologies used

  • MarkUp language : HTML
  • Styling language : CSS
  • Scripting language : JavaScript
  • Server Side Language : Django
  • Responsiveness : BootStrap
  • Version Control : GitHub
  • OS : Ubuntu 16.04
  • Report Writing : LaTex
